# Mentorist

Интерактивный demo-кабинет сервиса личной стратегии, наставничества и групповой поддержки. Приложение собрано как кликабельный Next.js-прототип, но уже ведет себя как полноценный тестовый продукт: цели можно создавать и редактировать, этапы закрывать, сообщения отправлять, посты публиковать, а все изменения сохраняются локально.

## Что внутри

- Связный кабинет с модулями `Профиль`, `Формер`, `Цели`, `Конструктор`, `Контакты`, `Лента`, `Группы`
- Интерактивное колесо баланса по 4 сферам
- Планнер недели с этапами целей и быстрыми задачами
- Рабочий конструктор целей с тестовыми данными
- Сеть контактов, чаты, перевод `менто`
- Группы поддержки и общая лента постов
- Pan/zoom-канвасы для карт целей, смыслов и контактов

## Стек

- Next.js 16
- React 19
- CSS в `app/globals.css` без отдельной UI-библиотеки
- Локальное сохранение состояния через `localStorage`

## Команды

```bash
npm run dev
npm run build
```

## Структура

- `app/` — layout, entrypoint, глобальные стили, иконка
- `components/mentorist-prototype.js` — основной app shell и бизнес-логика demo
- `components/mentorist-data.js` — тестовые данные, словари, дефолтное состояние
- `docs/design-system.md` — дизайн-система и визуальные правила

## Состояние и dev-инструменты

- Основное demo-состояние хранится в `localStorage` по ключу `mentorist-demo-state-v2`
- Выбранная шрифтовая пара хранится по ключу `mentorist-font-preset-v2`
- Базовая пара интерфейса: `Georgia / Avenir`
- Скрытый тест шрифтов оставлен для dev-проверок:
  - `Alt+Shift+F`
  - `?fontlab=1` в URL
